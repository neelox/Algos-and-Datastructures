We follow some steps to complete any type of task successfully.
In coding, we follow some steps (like what data goes where etc.) to write code to solve a problem. The ordered collection(from first to last) of these steps is called an Algorithm. There may be more than one algorithms to complete a task. Think!
When we write code we have to play around with data. So we devise wiser ways to handle that data, to store it in our computer's memory. In other words we handle data by making some rules about what data goes where in the memory and how it is accessed from that memory. We use different data Structures for differnet problems.
